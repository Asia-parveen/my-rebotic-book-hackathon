# Module 3: NVIDIA Isaac & Isaac ROS Quiz

This quiz will test your understanding of the core concepts covered in Module 3: NVIDIA Isaac & Isaac ROS.

---

1.  What is the primary purpose of Isaac Sim in the NVIDIA Isaac ecosystem?
    A) To provide GPU-accelerated ROS packages
    B) To offer a high-fidelity simulation environment for robotics
    C) To serve as a hardware platform for edge computing
    D) To function as a deep learning training framework
    **Answer:** B
    **Explanation:** Isaac Sim is NVIDIA's advanced robotics simulator that provides photorealistic simulation capabilities powered by RTX technology and PhysX physics engine for testing and validating robotic algorithms.

2.  Which NVIDIA technology does Isaac Sim use for its physics simulation?
    A) CUDA
    B) PhysX
    C) Tensor Cores
    D) RTX
    **Answer:** B
    **Explanation:** Isaac Sim uses NVIDIA's PhysX engine for accurate rigid body dynamics and realistic physics simulation.

3.  What does NITROS stand for in the Isaac ROS ecosystem?
    A) NVIDIA Integrated Toolkit for Robotic Systems
    B) Network Interface for Transforming Robotics Sensor Data
    C) NVIDIA Intelligent Transport and Routing System
    D) Next-Generation Intelligent Robotic Operating System
    **Answer:** B
    **Explanation:** NITROS stands for Network Interface for Transforming Robotics Sensor Data, which optimizes data transfer between Isaac ROS nodes.

4.  Which Isaac ROS package provides GPU-accelerated fiducial marker detection?
    A) Isaac ROS Image Pipeline
    B) Isaac ROS Stereo DNN
    C) Isaac ROS AprilTag
    D) Isaac ROS RealSense
    **Answer:** C
    **Explanation:** Isaac ROS AprilTag provides GPU-accelerated fiducial marker detection capabilities.

5.  What is the maximum AI performance of the Jetson AGX Orin platform?
    A) 32 TOPS
    B) 21 TOPS
    C) 275 TOPS
    D) 1.33 TFLOPS
    **Answer:** C
    **Explanation:** The Jetson AGX Orin platform provides up to 275 TOPS of AI performance, making it the most powerful Jetson device.

6.  Which computer vision library is part of Isaac ROS for GPU-accelerated image processing?
    A) OpenCV
    B) CV-CUDA
    C) VPI
    D) Both B and C
    **Answer:** D
    **Explanation:** Both CV-CUDA (Computer Vision CUDA) and VPI (Vision Programming Interface) are part of Isaac ROS for GPU-accelerated computer vision operations.

7.  What is the primary advantage of using Isaac ROS over traditional ROS packages?
    A) Lower cost
    B) GPU acceleration for improved performance
    C) Simpler API
    D) Better documentation
    **Answer:** B
    **Explanation:** The primary advantage of Isaac ROS is GPU acceleration, which provides significantly improved performance for perception, navigation, and manipulation tasks.

8.  Which file format serves as the foundation for scene representation in Isaac Sim?
    A) URDF
    B) SDF
    C) USD (Universal Scene Description)
    D) OBJ
    **Answer:** C
    **Explanation:** USD (Universal Scene Description) serves as the foundation for scene representation in Isaac Sim, enabling scene composition and asset sharing.

9.  What is a key feature of the Vision Programming Interface (VPI) in Isaac ROS?
    A) It only works with CUDA
    B) It provides a unified interface across different accelerators
    C) It is only for image processing
    D) It replaces OpenCV completely
    **Answer:** B
    **Explanation:** VPI provides a unified interface for vision algorithms that works across different accelerators including CUDA, OpenCV, and other platforms.

10. Which Jetson platform is recommended for entry-level AI robotics applications?
    A) Jetson AGX Orin
    B) Jetson Xavier NX
    C) Jetson TX2
    D) Jetson Nano
    **Answer:** D
    **Explanation:** The Jetson Nano is the entry-level platform in the Jetson family, providing 472 GFLOPS AI performance suitable for basic AI robotics applications.

11. What does the acronym TOPS stand for in the context of AI performance metrics?
    A) Terabytes of Processing Storage
    B) Trillions of Operations Per Second
    C) Total Operations Per System
    D) Terabytes of Processed Sensors
    **Answer:** B
    **Explanation:** TOPS stands for Trillions of Operations Per Second, a common metric for measuring AI performance, particularly for neural network inference.

12. Which Isaac ROS package is specifically designed for GPU-accelerated navigation?
    A) Isaac ROS Bitmapped Navigation
    B) Isaac ROS Path Planning
    C) Isaac ROS Navigation2
    D) Isaac ROS Motion Planning
    **Answer:** A
    **Explanation:** Isaac ROS Bitmapped Navigation is specifically designed as a GPU-accelerated navigation stack.
